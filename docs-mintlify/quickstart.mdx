## Setup your environment

Get your API key from the [Jean Memory Dashboard](https://jeanmemory.com/dashboard).

<Info>
  **Prerequisite** You should have created a Jean Memory account. If you haven't, [sign up here](https://jeanmemory.com/auth).
</Info>

## Choose your framework

<Tabs>
  <Tab title="React">
    ### Install the SDK
    
    <CodeGroup>
    ```bash npm
    npm install @jeanmemory/react @assistant-ui/react
    ```
    
    ```bash yarn
    yarn add @jeanmemory/react @assistant-ui/react
    ```
    
    ```bash pnpm
    pnpm add @jeanmemory/react @assistant-ui/react
    ```
    </CodeGroup>
    
    ### Add to your app
    
    ```javascript
    import { useJeanAgent, SignInWithJean, JeanChat } from '@jeanmemory/react';
    
    function App() {
      const { agent, signIn } = useJeanAgent({
        apiKey: process.env.REACT_APP_JEAN_API_KEY,
        systemPrompt: "You are a helpful assistant"
      });
    
      if (!agent) return <SignInWithJean onSuccess={signIn} />;
      return <JeanChat agent={agent} />;
    }
    ```
    
    ### Environment variables
    
    Create a `.env` file in your project root:
    
    ```bash
    REACT_APP_JEAN_API_KEY=jean_sk_your_api_key_here
    ```
  </Tab>
  
  <Tab title="Python">
    ### Install the SDK
    
    ```bash
    pip install jeanmemory
    ```
    
    ### Create your agent
    
    ```python
    from jeanmemory import JeanAgent
    import os
    
    agent = JeanAgent(
        api_key=os.getenv("JEAN_API_KEY"),
        system_prompt="You are a helpful assistant",
        modality="chat"
    )
    
    agent.run()
    ```
    
    ### Environment variables
    
    Create a `.env` file in your project root:
    
    ```bash
    JEAN_API_KEY=jean_sk_your_api_key_here
    ```
  </Tab>
  
  <Tab title="Node.js">
    ### Install the SDK
    
    <CodeGroup>
    ```bash npm
    npm install @jeanmemory/node
    ```
    
    ```bash yarn
    yarn add @jeanmemory/node
    ```
    
    ```bash pnpm
    pnpm add @jeanmemory/node
    ```
    </CodeGroup>
    
    ### Create your agent
    
    ```javascript
    import { JeanAgent } from '@jeanmemory/node';
    import dotenv from 'dotenv';
    
    dotenv.config();
    
    const agent = new JeanAgent({
      apiKey: process.env.JEAN_API_KEY,
      systemPrompt: "You are a helpful assistant"
    });
    
    await agent.run();
    ```
    
    ### Environment variables
    
    Create a `.env` file in your project root:
    
    ```bash
    JEAN_API_KEY=jean_sk_your_api_key_here
    ```
  </Tab>
  
  <Tab title="cURL">
    ### Make your first request
    
    ```bash
    curl -X POST https://api.jeanmemory.com/mcp/messages/ \
      -H "X-Api-Key: jean_sk_your_api_key_here" \
      -H "Content-Type: application/json" \
      -d '{
        "jsonrpc": "2.0",
        "method": "tools/call",
        "params": {
          "name": "add_memories",
          "arguments": {
            "text": "My favorite color is blue",
            "tags": ["personal", "preferences"]
          }
        },
        "id": "1"
      }'
    ```
  </Tab>
</Tabs>
