## Quick Start

The fastest way to add Jean Memory to your React app:

```javascript
import { useJeanAgent, JeanChat } from '@jeanmemory/react';

function App() {
  const { agent } = useJeanAgent({ 
    apiKey: process.env.REACT_APP_JEAN_API_KEY 
  });
  
  return <JeanChat agent={agent} />;
}
```

That's literally it. You now have a full chat interface with persistent memory.

## The Hook

### `useJeanAgent`

The main hook that manages everything:

```javascript
const {
  agent,           // The AI agent instance
  signIn,          // Function to authenticate
  signOut,         // Function to sign out
  isAuthenticated, // Boolean auth state
  isLoading,       // Loading state
  error           // Error state
} = useJeanAgent(options);
```

### Options

```javascript
useJeanAgent({
  apiKey: "jean_sk_...",        // Required: Your API key
  systemPrompt: "You are...",   // Optional: Custom AI personality
  baseUrl: "https://...",       // Optional: Custom API endpoint
  autoConnect: true,            // Optional: Auto-connect on mount
  debug: false                  // Optional: Enable debug logging
})
```

## Common Patterns

### With Authentication

```javascript
function App() {
  const { agent, signIn, isAuthenticated } = useJeanAgent({
    apiKey: process.env.REACT_APP_JEAN_API_KEY
  });
  
  if (!isAuthenticated) {
    return (
      <div className="auth-container">
        <h1>Welcome</h1>
        <SignInWithJean onSuccess={signIn} />
      </div>
    );
  }
  
  return <JeanChat agent={agent} />;
}
```

### Custom UI

Don't want our UI? Build your own:

```javascript
function CustomChat() {
  const { agent } = useJeanAgent({ apiKey: "..." });
  const [message, setMessage] = useState('');
  const [response, setResponse] = useState('');
  
  const handleSend = async () => {
    const result = await agent.sendMessage(message);
    setResponse(result.content);
  };
  
  return (
    <div>
      <input 
        value={message} 
        onChange={(e) => setMessage(e.target.value)}
      />
      <button onClick={handleSend}>Send</button>
      <div>{response}</div>
    </div>
  );
}
```

### With Next.js

```javascript
// app/chat/page.js

```

## Memory Management

### Store Memories

```javascript
// Memories are automatically stored from conversations
// But you can also manually add them:
await agent.addMemory("User prefers dark mode");
await agent.addMemory("Working on Project Apollo", ["work", "apollo"]);
```

### Search Memories

```javascript
const memories = await agent.searchMemories("project deadline");
console.log(memories);
```

### Clear Memories

```javascript
await agent.clearMemories(); // Use with caution!
```

## Error Handling

```javascript
function App() {
  const { agent, error, isLoading } = useJeanAgent({
    apiKey: process.env.REACT_APP_JEAN_API_KEY
  });
  
  if (isLoading) return <div>Connecting to Jean...</div>;
  if (error) return <div>Error: {error.message}</div>;
  if (!agent) return <div>Initializing...</div>;
  
  return <JeanChat agent={agent} />;
}
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Agent is undefined">
    Make sure your API key is set correctly in your environment variables.
  </Accordion>
  
  <Accordion title="CORS errors">
    The SDK handles CORS automatically. If you see CORS errors, check that you're using the SDK methods, not direct API calls.
  </Accordion>
  
  <Accordion title="Memory not persisting">
    Ensure you're using the same API key across sessions. Each API key has its own memory space.
  </Accordion>
  
  <Accordion title="Styling issues">
    The JeanChat component uses CSS-in-JS. If you see styling conflicts, wrap it in a container with `isolation: isolate`.
  </Accordion>
