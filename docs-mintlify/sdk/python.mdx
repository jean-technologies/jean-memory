pip install jeanmemory

## The 5-Line Wonder

```python
from jeanmemory import JeanAgent

agent = JeanAgent(api_key="jean_sk_...")
agent.run()
```

That's it. Your Python app now has persistent memory.

## Configuration

```python
agent = JeanAgent(
    api_key="jean_sk_...",           # Required
    system_prompt="You are...",      # Optional: AI personality
    modality="chat",                  # "chat" or "completion"
    model="gpt-4",                    # Model to use
    temperature=0.7,                  # Response creativity
    max_tokens=None,                  # Response length limit
    debug=False                       # Enable debug logging
)
```

## Usage Patterns

### Interactive Chat

```python
from jeanmemory import JeanAgent

agent = JeanAgent(api_key="jean_sk_...")
agent.run()  # Starts interactive chat with memory
```

### Programmatic API

```python
# Send a message
response = agent.send_message("What's on my todo list?")
print(response.content)

# With streaming
for chunk in agent.send_message_stream("Tell me about the project"):
    print(chunk, end="")
```

### Memory Management

```python
# Add memories with tags
agent.add_memory(
    text="Meeting with Sarah about Q4 roadmap",
    tags=["meetings", "q4", "roadmap"]
)

# Search memories
memories = agent.search_memories("roadmap")

# Get all memories
all_memories = agent.get_memories()

# Delete specific memory
agent.delete_memory(memory_id="mem_123")
```

## Async Support

```python
import asyncio
from jeanmemory import AsyncJeanAgent

async def main():
    agent = AsyncJeanAgent(api_key="jean_sk_...")
    
    response = await agent.send_message("Hello")
    print(response.content)
    
    # Async streaming
    async for chunk in agent.send_message_stream("Tell me more"):
        print(chunk, end="")

asyncio.run(main())
```

## Integration Examples

### Flask Application

```python
from flask import Flask, request, jsonify
from jeanmemory import JeanAgent

app = Flask(__name__)
agent = JeanAgent(api_key="jean_sk_...")

@app.route('/chat', methods=['POST'])
def chat():
    message = request.json['message']
    response = agent.send_message(message)
    return jsonify({'response': response.content})
```

### Discord Bot

```python
import discord
from jeanmemory import JeanAgent

agent = JeanAgent(api_key="jean_sk_...")
client = discord.Client()

@client.event
async def on_message(message):
    if message.author == client.user:
        return
    
    response = agent.send_message(message.content)
    await message.channel.send(response.content)

client.run('discord_token')
```

### Jupyter Notebook

```python
from jeanmemory import JeanAgent
from IPython.display import Markdown, display

agent = JeanAgent(api_key="jean_sk_...")

def chat(message):
    response = agent.send_message(message)
    display(Markdown(response.content))

# Now use it
chat("What did we work on last session?")
```

## Advanced Features

### Custom Memory Processing

```python
agent = JeanAgent(
    api_key="jean_sk_...",
    memory_processor=lambda text: text.upper(),  # Custom processing
    memory_filter=lambda memory: "private" not in memory.tags
)
```

### Conversation Management

```python
# Start new conversation
agent.new_conversation()

# Get conversation history
history = agent.get_conversation_history()

# Export conversations
agent.export_conversations("conversations.json")
```

### Error Handling

```python
from jeanmemory import JeanAgent, JeanError

try:
    agent = JeanAgent(api_key="jean_sk_...")
    response = agent.send_message("Hello")
except JeanError as e:
    print(f"Jean Memory error: {e}")
```

## CLI Usage

Jean Memory SDK includes a CLI:

```bash
# Start interactive chat
jean chat --api-key jean_sk_...

# Add memory from command line
jean remember "Important meeting tomorrow at 3pm"

# Search memories
jean search "meeting"
```
