## What is Jean Memory?

Jean Memory is a universal memory layer for AI applications. Add persistent memory to any AI in just 5 lines of code.

## The 5-Line Integration

<CodeGroup>
```javascript React
import { useJeanAgent, JeanChat } from '@jeanmemory/react';

function App() {
  const { agent } = useJeanAgent({ apiKey: "jean_sk_..." });
  return <JeanChat agent={agent} />;
}
```

```python Python
from jeanmemory import JeanAgent

agent = JeanAgent(api_key="jean_sk_...")
agent.run()
```
</CodeGroup>

## Key Features

- **Universal** - Works with ChatGPT, Claude, Cursor, and custom apps
- **Persistent** - Memories survive across sessions
- **Intelligent** - AI-powered retrieval, not just keyword matching
- **Simple** - No database setup, no schemas, just an API key

## How It Works

1. **Store** - Conversations and context are automatically saved
2. **Retrieve** - Relevant memories are pulled based on context
3. **Enhance** - AI responses include personal context
