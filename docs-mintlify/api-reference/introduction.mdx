## Base URL

All API requests should be made to:

```
https://api.jeanmemory.com
```

## Authentication

The Jean Memory API uses API keys for authentication. Include your API key in the `X-Api-Key` header:

```bash
X-Api-Key: jean_sk_your_api_key_here
```

<Warning>
  Never expose your API key in client-side code or public repositories. Always use environment variables.
</Warning>

## Request Format

All API endpoints accept JSON-RPC 2.0 formatted requests:

```json
{
  "jsonrpc": "2.0",
  "method": "tools/call",
  "params": {
    "name": "tool_name",
    "arguments": {
      // tool-specific arguments
    }
  },
  "id": "unique-request-id"
}
```

## Response Format

Successful responses return:

```json
{
  "jsonrpc": "2.0",
  "result": {
    // tool-specific response
  },
  "id": "unique-request-id"
}
```

Error responses return:

```json
{
  "jsonrpc": "2.0",
  "error": {
    "code": -32603,
    "message": "Error description"
  },
  "id": "unique-request-id"
}
```
