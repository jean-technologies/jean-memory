## Prerequisites

Before you begin, make sure you have:

- Node.js 18+ or Python 3.8+ installed
- A Jean Memory account ([sign up here](https://jeanmemory.com/auth))
- Your API key from the dashboard

## Local Development

<Tabs>
  <Tab title="React">
    ### Create a new React app
    
    ```bash
    npx create-react-app my-jean-app
    cd my-jean-app
    ```
    
    ### Install Jean Memory
    
    ```bash
    npm install @jeanmemory/react @assistant-ui/react
    ```
    
    ### Set up environment variables
    
    Create `.env.local`:
    ```bash
    REACT_APP_JEAN_API_KEY=jean_sk_your_api_key_here
    ```
    
    ### Start development server
    
    ```bash
    npm start
    ```
  </Tab>
  
  <Tab title="Next.js">
    ### Create a new Next.js app
    
    ```bash
    npx create-next-app@latest my-jean-app
    cd my-jean-app
    ```
    
    ### Install Jean Memory
    
    ```bash
    npm install @jeanmemory/react @assistant-ui/react
    ```
    
    ### Set up environment variables
    
    Create `.env.local`:
    ```bash
    NEXT_PUBLIC_JEAN_API_KEY=jean_sk_your_api_key_here
    ```
    
    ### Start development server
    
    ```bash
    npm run dev
    ```
  </Tab>
  
  <Tab title="Python">
    ### Create virtual environment
    
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows: venv\Scripts\activate
    ```
    
    ### Install Jean Memory
    
    ```bash
    pip install jeanmemory python-dotenv
    ```
    
    ### Set up environment variables
    
    Create `.env`:
    ```bash
    JEAN_API_KEY=jean_sk_your_api_key_here
    ```
    
    ### Create your first agent
    
    ```python
    from jeanmemory import JeanAgent
    from dotenv import load_dotenv
    import os
    
    load_dotenv()
    
    agent = JeanAgent(
        api_key=os.getenv("JEAN_API_KEY"),
        system_prompt="You are a helpful assistant"
    )
    
    agent.run()
    ```
  </Tab>
</Tabs>

## Common Issues

<Warning>
  **CORS Errors:** API keys should only be used in server-side code. For client-side apps, use our SDK which handles authentication securely.
</Warning>

<Info>
  **Rate Limits:** Development keys have lower rate limits. Upgrade to Pro for production use.
</Info>
