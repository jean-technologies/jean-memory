## Two Paths to Get Started

There are two primary ways to get value from Jean Memory in under 5 minutes. Choose the path that fits your goal.

<CardGroup cols={2}>
  <Card
    title="Path A: Add a Chatbot UI"
    icon="rocket"
  >
    If you want to quickly add a complete, pre-built chat interface to your React application.
    
    ```tsx
    import { JeanAgent } from 'jeanmemory-react';

    function App() {
      return <JeanAgent apiKey="..." />;
    }
    ```
  </Card>
  <Card
    title="Path B: Add Context to Your Backend"
    icon="layer-group"
  >
    If you have your own AI agent and just want to inject our context into your existing LLM calls.
    
    ```typescript
    // In your backend API route:
    import { JeanMemory } from '@jeanmemory/node';
    
    const jean = new JeanMemory({ apiKey: "..." });
    
    const { context } = await jean.getContext({
      token: userToken,
      message: userMessage
    });
    ```
  </Card>
</CardGroup>
