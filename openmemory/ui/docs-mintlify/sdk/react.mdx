## Quick Start: The `<JeanAgent />` Component

This is the simplest way to add a complete, memory-enabled AI chatbot to your application. It handles authentication, UI, and state management automatically.

```tsx {{ title: 'src/App.tsx' }}
import { JeanAgent } from 'jeanmemory-react';
import 'jeanmemory-react/dist/style.css'; // Optional: for default styling

function App() {
  return (
    <div style={{ height: '600px', width: '400px', margin: 'auto' }}>
      <JeanAgent apiKey="YOUR_JEAN_SK_API_KEY" />
    </div>
  );
}
```

This single component provides a secure "Sign In with Jean" button and a beautiful, production-ready chat interface.

## Advanced Usage: The `useJeanAgent` Hook

For developers who need to build a custom UI, the `useJeanAgent` hook is the core of the SDK. It provides all the necessary state and logic, giving you complete control over the user experience.

```tsx {{ title: 'src/MyCustomChat.tsx' }}
import { useJeanAgent, SignInWithJean } from 'jeanmemory-react';

function CustomChatExperience() {
  const agent = useJeanAgent({
    apiKey: "YOUR_JEAN_SK_API_KEY"
  });

  if (!agent.isAuthenticated) {
    // The `<SignInWithJean />` component handles the entire OAuth flow.
    // The `onSuccess` callback receives the user object, which we use
    // to update the agent's state.
    return (
      <SignInWithJean
        apiKey={agent.config.apiKey}
        onSuccess={(user) => agent.setUser(user)}
      />
    );
  }

  // Build your custom chat UI using agent state and methods
  return (
    <div>
      {/* Your custom UI here... */}
    </div>
  );
}
```

### Hook Return Object

The `agent` object returned by `useJeanAgent` contains everything you need:

| Property                | Type                                     | Description                                                 |
| ----------------------- | ---------------------------------------- | ----------------------------------------------------------- |
| `isAuthenticated`       | `boolean`                                | True if the user is authenticated.                          |
| `user`                  | `JeanUser | null`                        | The authenticated user object.                              |
| `messages`              | `Message[]`                              | An array of the current conversation messages.              |
| `isLoading`             | `boolean`                                | True when a message is being processed.                     |
| `error`                 | `string | null`                         | Any error messages from the API.                            |
| `sendMessage(msg)`      | `(msg: string) => Promise<void>`         | Sends a message to the assistant.                           |
| `connect(service)`      | `(service: 'notion') => Promise<string>` | Initiates the connection flow for an external service.      |
| `setUser(user)`         | `(user: JeanUser) => void`               | Manually sets the user state (used by `SignInWithJean`).    |
| `signOut()`             | `() => void`                             | Signs the user out and clears their state.                  |

## TypeScript Support

The SDK is written in TypeScript and exports all relevant types for a fully typed development experience.

```typescript
import { useJeanAgent, JeanUser, JeanAgentConfig } from 'jeanmemory-react';

const config: JeanAgentConfig = {
  apiKey: "jean_sk_...",
};

const agent = useJeanAgent(config);

if (agent.user) {
  console.log(agent.user.email); // string
}
```
